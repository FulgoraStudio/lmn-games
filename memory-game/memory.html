Memoria

Memoria es regresar al pasado para traer al presente la antigua sabiduría que crece generación tras generación. Este juego está inspirado en el arte rupestre de Colo Michi Có, en el departamento de Minas, y busca homenajear a los pueblos originarios neuquinos.

Errores: 0

Instrucciones

Memoria es el juego donde debés encontrar los pares de ilustraciones idénticas grabadas sobre piedras antiguas.

Ganás cuando descubrís todos los pares. ¡Tratá de memorizar tu posición al principio de cada partida!
